bzoj
|
|-- bzoj.cpp 程序文件
|-- data.dat bzoj题库数据
|-- done.txt AC过的题，初始可以把所有A过的题粘进去，正常退出的话自动维护。
|-- black.txt 黑名单。选题时会跳过。错题、神题、没题面、不想做等等。


初始输入一个 (0,1) 的实数，代表难度（越小越难）。
然后输入1,2,3分别代表想做老题，新题，both。
选题算法含有随机因素。可能会出现高于输入难度的题。
大部分情况与本题AC率与AC人数和输入的难度有关。

可以把程序路径放入path,然后就可以直接命令行运行了。
